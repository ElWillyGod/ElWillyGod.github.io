IM - Project

El proyecto consiste en la evaluación del componente Workflow de Oracle Apex, para su posible implementación en la gestión de trámites simples, algunos de ustedes se preguntaran que es Apex, Oracle Apex es un herramienta low code para el desarrollo de aplicaciones web con soporte de su propia base de datos integrada.

Beneficios

  • Facilidad de modelar.
  • Permite manejar roles.
  • Automatizar fácilmente tareas.
  • Realizar desde procesos mas simples a mas complejos.
  • Monitoreo sobre el flujo.
  • Bajo costo de implementacion.
  • Notificacion y recordatorio via email.
  • Facil modificacion y mantenimiento.
  • Accede a este link para ver el documento consultoria

Contras

  • Interfaz poco intuitiva.
  • Requiere conocimiento en PL/SQL.
  • Curva de aprendizaje media.
  • Documentacion e informacion limitada
  • Modificacion fuera del entorno de desarrollo
  • Las Tablas de base de datos que se utilizan para guardar la informacion son inaccesibles directamente.
